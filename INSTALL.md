# Install, 3 steps

1. In your profile repo (github.com/rayidali/rayidali), replace the assets folder contents with the SVGs in this kit. Same filenames, drop in.
2. Replace README.md with the one in this kit.
3. Commit and push. Done.

Note: everything is raw SVG, edit any text by opening the file. The pixel name is generated rect art, so to change it, ask Claude to regenerate rather than hand editing.
